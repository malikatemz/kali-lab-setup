# 🔐 Kali Linux Lab Setup

Automated setup script and reference guides for a professional pentest lab.

## Setup
```bash
chmod +x setup.sh
./setup.sh
```

## What's Installed
- **Scanning:** nmap, masscan, nikto, gobuster
- **Exploitation:** Metasploit, SQLmap, Hydra
- **Traffic analysis:** Wireshark, tcpdump
- **Password cracking:** John, Hashcat
- **Python:** impacket, scapy, pwntools

## Lab Structure
```
~/lab/
  targets/web/
  targets/network/
  targets/active_directory/
  exploits/
  reports/
  loot/
```
